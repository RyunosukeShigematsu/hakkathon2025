フォルダ,ファイル名,役割
customer/,app.js,共有ロジック。店側の在庫を減らす処理や年齢レンジ判定を搭載。
,register.html,ユーザーの属性（性別・好み・年齢）を登録。
,receipt.html,レシート保存時に「やり直し権利」を計算。
,coupons.html,在庫数と連動したリアルタイムなクーポン提示。
,home.html,獲得したクーポンの管理。
shop/,index.html,管理ダッシュボード。在庫数やログ件数を一目で確認。
,create_coupon.html,（今回完成） クーポンの編集・追加発行・年齢レンジ設定。
,analysis.html,蓄積された行動ログに基づいた需要予測・ランキング表示。
